  ## What is an accessibility enhancement for an `<img>` element?

  </template>

  <template slot='categories'>
  
  <Category slug='html'/>

  </template>

  <template slot='difficulty'>

  <Difficulty rating='easy' />  

  </template>

  <template slot='answer'>

  The `alt` tag should be set to a concise description of the image, not a single word file name of the image.
  
  For example:

  ```html

  <img src="luke-skywalker.jpg" alt="Luke Skywalker stands on his home planet of Tatooine">

  ```

  </template>
