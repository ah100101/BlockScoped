  ## What is the difference between `Object.create` and `Object.assign`?

  <template slot='answer'>

The `Object.create()` method creates a new object, using an existing object as the *prototype* of the newly created object. Where The `Object.assign()` method is used to copy the values of all enumerable own properties to a target object.
